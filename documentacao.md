# ⚡ BIRL Cheat Sheet: Referência Rápida

Para detalhes profundos, consulte a **[Documentação Completa no README](./README.md)**.

---

## 🏗️ Estrutura Base
- **Início**: `HORA DO SHOW`
- **Fim/Bloco**: `BIRL`

## 🧬 Variáveis e Tipos
- **Declaração**: `MONSTRO X = 10;`
- **Atribuição**: `X = 20;`
- **Arrays**: `MONSTRO LISTA = VEM MONSTRO;`

## 🗣️ I/O (Entrada e Saída)
- **Print**: `CE QUER VER ESSA PORRA? ("Texto: %s\n", VAR);`
- **Scan**: `QUE QUE CE QUER MONSTRAO? ("%d", &IDADE);`

## 🔀 Controle de Fluxo
```birl
ELE QUE A GENTE QUER? (X > 10)
    // IF
QUE NAO VAI DAR O QUE? (X > 5)
    // ELSE IF
NAO VAI DAR NAO
    // ELSE
BIRL
```

## 🔄 Loops
- **While**: `NEGATIVA BAMBAM (X < 10) ... BIRL`
- **For**: `MAIS QUERO MAIS (I=0; I<10; I++) ... BIRL`

## 🏛️ Orientação a Objetos (OOP)
- **Definir Jaula**: `JAULA Nome ... BIRL`
- **Instanciar**: `MONSTRO OBJ = SAINDO DA JAULA Nome;`
- **Método**: `OH O HOME AI PO (MONSTRO NOME(args)) ... BIRL`
- **Contexto**: `O_PAI.atributo = valor;`
- **Chamar**: `AJUDA O MALUCO TA DOENTE OBJ.METODO();`

## 📦 Métodos de Lista
- `.BOTA_PRA_FUDER(val)` -> `push`
- `.TAMANHO()` -> `length`
- `.PEGA(index)` -> `get`
