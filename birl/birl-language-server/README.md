# 🧠 BIRL Language Server (LSP)

Este é o cérebro da inteligência de código BIRL. Ele fornece autocompletar e validação em tempo real para o VS Code.

## 🚀 Como funciona

O servidor é iniciado automaticamente pela extensão do VS Code. Ele utiliza o protocolo LSP para se comunicar com o editor.

## 🛠️ Funcionalidades
- **Autocomplete**: Sugere palavras-chave da linguagem (JAULA, MONSTRO, BIRL, etc).
- **Diagnostics**: Avisa se você esqueceu de fechar um bloco com `BIRL` (Validação de shape).
