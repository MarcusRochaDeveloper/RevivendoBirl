# 🔌 BIRL VS Code Extension

A ponte definitiva entre o seu código e a glória. Esta extensão integra o **LSP** e o **Runtime** no seu editor.

## 🚀 Como testar

1.  Abra a pasta `birl/birl-vscode-extension` no VS Code.
2.  Pressione **F5** (Extension Development Host).
3.  Abra um arquivo `.birl` e aproveite o suporte completo!

## 🛠️ Comandos
- **Ctrl + Alt + B**: Executa o treino usando o `birl-runtime`.
