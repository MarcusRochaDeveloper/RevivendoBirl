# 🏃 BIRL Runtime (Interpretador CLI)

Este módulo contém o coração da linguagem BIRL. Ele é responsável por carregar os arquivos `.birl` e executar a lógica maromba.

## 🚀 Como usar

Para rodar um arquivo manualmente via terminal:

1.  Acesse esta pasta: `cd birl-runtime`
2.  Execute via Node.js:
    ```bash
    node bin/birl.js ../../programa.birl
    ```

## 🛠️ Funcionalidades
- Suporte a **JAULAS** (Classes).
- Manipulação de listas dinâmicas com **VEM MONSTRO**.
- Gerenciamento de contexto local com **O_PAI**.
