# 🏋️‍♂️ BIRL Monorepo: O Ecossistema Pro do Bodybuilder

Este repositório contém a suíte completa de ferramentas para a linguagem **BIRL (Binary Interactive Runtime Language)**. 

## 🏗️ Estrutura

- **[birl-runtime](./birl-runtime)**: Interpretador CLI para executar arquivos `.birl`.
- **[birl-language-server](./birl-language-server)**: LSP para inteligência de código (Autocomplete e Diagnósticos).
- **[birl-vscode-extension](./birl-vscode-extension)**: Extensão VS Code que integra tudo.

## 🚀 Como Iniciar

1.  **Instalação**: 
    Acesse a pasta raiz e instale as dependências:
    ```bash
    npm run install:all
    ```

2.  **Desenvolvimento**:
    Abra a pasta `birl/birl-vscode-extension` no VS Code e pressione **F5** para testar.

## 🏆 Objetivo
Transformar o meme em uma plataforma de desenvolvimento funcional, robusta e escalável.
