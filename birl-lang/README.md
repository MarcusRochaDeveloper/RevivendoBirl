# Extension: BIRL Language Support

Esta extensão traz o suporte oficial (e monstro) para a linguagem **BIRL (Binary Interactive Runtime Language) OOP Edition** no seu VS Code.

## 🚀 Funcionalidades

- **Syntax Highlighting**: Colorização completa para comandos, classes (`JAULA`), variáveis (`MONSTRO`) e funções.
- **Execução Direta**: Execute seus treinos instantaneamente no terminal dedicado do VS Code.
- **Keybinding**: Atalho rápido para quem não tem tempo a perder.

## 🏋️‍♂️ Como Usar

1.  Abra qualquer arquivo com a extensão `.birl`.
2.  Pressione `Ctrl + Alt + B` (ou `Cmd + Alt + B` no Mac).
3.  Acompanhe o crescimento do seu código no OutputChannel **BIRL**.

## 🛠️ Comandos Disponíveis

- `bir.run`: Executa o arquivo `.birl` atual usando o interpretador `index.js` localizado na raiz do seu workspace.
